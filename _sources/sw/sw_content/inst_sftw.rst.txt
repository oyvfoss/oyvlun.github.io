Installing software
===================

Anything that is a part of the Ubuntu (Mint?) distribution can be downloaded directly from the Software Manager. Other packages typically need to be installed from a *tarball*.

Installing packages in the distribution
+++++++++++++++++++++++++++++++++++++++

Can be done through the software manager. Alternatively:

::

 sudo apt-get install packagename


Installing from tarballs
++++++++++++++++++++++++

Taken from `this page <http://www.cyberciti.biz/faq/install-tarballs/>`_.

Tarballs are in one of the following formats: *.tar.gz, .tgz or .tar.bz2*.

There are three steps to installing from  process:

**1: Uncompress**

Execute one of these (depending on the extension).

::

 $ tar zxf file.tar.gz
 $ tar zxf file.tgz
 $ tar jxf file.tar.bz2
 $ tar jxf file.tbz2
