Updating (Git) 
==============

Example from updating the *Basemap* module:

Repositories in */usr/local/src/ziptar/repos/basemap* (github)..

**sudo git fetch origin** : "Status" command, seeing what has changed in the exetrnal "master" directory. In this case: gives a list of versions newer then my current one.

**git checkout v1.0.7rel** : Pulling the version I want.

**sudo python setup.py build** : Building basemap module with new version.
