Maps
++++

Using the *Basemap* module. There is a *pycurrents.mapper* but I can't figure out where the documentation is.

Latitude is [-90,90]. Longitude can be between -360 and +720. 

::

 from mpl_toolkits.basemap import Basemap
 m = Basemap(projection = 'cyl',llcrnrlon=min_lon,llcrnrlat=min_lat,urcrnrlon=max_lon,urcrnrlat=max_lat)
 m.fillcontinents(alpha = 1)
 m.drawcoastlines()
 if ann=='n': fcolor ='#99ffff'
 else: fcolor ='w'
 m.drawmapboundary(fcolor)
 m.drawmeridians(np.arange(min_lon,max_lon,2),labels=[0,0,0,1])
 m.drawparallels(np.arange(min_lat,max_lat,2),labels=[1,0,0,0])
 x,y = m(lons,lats)
