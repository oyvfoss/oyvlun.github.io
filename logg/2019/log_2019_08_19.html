Detiding in tsclass¶

Got stuck on tsclass.detide(). Problem seemed to be that the operation

t = t[~A.mask]
A = A[~A.mask]

in oyv.num.tides.tidefit_ts_1d() makes t:(1, N), A:(1, N) (adds an excessive dimension. Problem solved by rewriting to

t = np.ma.squeeze(t[~A.mask])
A = np.ma.squeeze(A[~A.mask])

Don’t really know why this happens, likely a python version issue..

Note: tsclass.detide() returns the tide fit as self.fu.

Note: Checked that spectra look fairly similar independent of what window shape (‘bman’, ‘han’, ‘none’ is used).
