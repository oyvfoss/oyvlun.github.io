subprocess i python

Ser ut som subprocess har blitt litt forenklet siden sist.

En enkel måte å utføre bash-kommandoen fra python er nå:

import subprocess
subprocess.run([kommando_streng], shell = True)

Master Bunch¶

Building a bunch:

  • Loading all fields.

  • Interpolate all time series onto an even grid spanning the 2013-2017 deployment, with spacing 1/(99 days).

  • Smoothing each time series with a 99 pt running window and subsampling each 99th datapoint - yielding a 1-day average dataset.

Results look sensible.

Script: build_master_bunch.py.

File: home/oyvindl/work/atwain/data/master_bunch.p.

Can update this script as I reprocedss or add new time series..
